# 个人投资平台 — 设计文档

## 定位

跑在个人电脑上的美股日线信号平台：每天收盘后自动拉数据、跑策略、推送买卖信号到 Telegram，本地 Streamlit 面板复盘。**只出信号，不自动下单**——省掉券商 API 对接与资金安全风险，人做最后一道审核。

核心决策：

| 决策项 | 选择 | 理由 |
|---|---|---|
| 市场 | 美股 | yfinance 免费日线数据充足 |
| 频率 | 日线，收盘后 | 每天跑一次，个人量化主流形态 |
| 标的 | 纯 ETF（21 只） | 技术面信号在 ETF 上远比个股可靠（个股受财报/新闻跳空干扰） |
| 通知 | Telegram + 本地面板 | 推送即时、复盘有图 |
| 部署 | 自己电脑 cron | 零成本 |

## Watchlist（config.yaml 可随时改）

- **大盘 ETF**：SPY、QQQ、IWM、DIA —— 大盘趋势背景板，趋势类信号最可靠
- **行业 ETF**：XLK、XLV、XLF、XLY、XLP、XLE、XLI、XLB、XLU、XLRE、XLC —— 动量轮动宇宙
- **热门主题**：SMH、SOXX（半导体，含 DRAM/存储厂商）、IGV（软件）
- **资产类**：TLT（长债）、GLD（黄金）、IBIT（比特币）—— 股债金币轮动观察

暂不含个股；科技敞口通过 QQQ/XLK/SMH 获得。

## 架构

```
┌─────────────┐   ┌──────────┐   ┌──────────────┐   ┌──────────────┐
│ 数据获取     │──▶│ 本地存储  │──▶│ 策略引擎      │──▶│ 信号生成      │
│ (yfinance)  │   │ (SQLite) │   │ (可插拔策略)  │   │ (落库+去重)   │
└─────────────┘   └────┬─────┘   └──────┬───────┘   └──────┬───────┘
                       │                │                   │
                       │         ┌──────▼───────┐   ┌───────▼──────┐
                       │         │ 回测引擎      │   │ Telegram 推送 │
                       │         └──────────────┘   └──────────────┘
                ┌──────▼───────────────────────────┐
                │ Streamlit 面板：信号/K线/回测       │
                └──────────────────────────────────┘
```

## 关键设计

- **prices 表**：`(symbol, date)` 主键，OHLCV + adj_close；fetcher 查库内最新日期只拉增量。yfinance 静默失败返回空表时，首次拉取报错而不是当成功。
- **signals 表**：`(date, symbol, strategy, direction)` 唯一约束实现幂等——重复运行不重复入库；`notified_at` 非空不重复推送。未配置 Telegram token 时打印到终端并视为已送达，网络失败才留待下次重试。
- **策略接口**：`Strategy.generate(prices: dict[symbol, DataFrame]) -> list[Signal]`，对全部历史出信号；每日运行筛选最新一天，回测用完整序列——同一套代码两处复用。
- **Signal.reason 用人话**：如 "SPY：20日均线上穿60日均线，收盘 $XXX"，推送与面板直接展示，方便人工审核。
- **回测**：信号驱动多头模拟（buy 全仓、sell 清仓），输出总收益/年化/最大回撤/夏普/胜率；Streamlit 回测页调用同一引擎。

## 内置策略

| 策略 | 逻辑 | 作用组 |
|---|---|---|
| sma_cross | 20 日均线上穿/下穿 60 日均线 | 大盘、主题、资产类 |
| momentum | 近 63 日收益组内排名，进入前 3 买入、跌出卖出 | 行业 ETF |
| rsi_reversal | RSI(14) 从超卖(<30)回升买入、从超买(>70)回落卖出 | 全部 |

## 风险与注意事项

- **yfinance 限流/改版**：数据层已抽象，可换 Alpha Vantage、Tiingo 等
- **过拟合**：回测好看 ≠ 未来赚钱；参数不调到极致，优先逻辑说得通的简单策略
- **电脑不开机**：`run_daily.py --date` 支持补跑，幂等保证补跑安全
- **信号仅供参考**：平台是提醒和纪律工具，不构成投资建议
